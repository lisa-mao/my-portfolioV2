# HII welcome to my portfolio made with Next.js!!

A clean, responsive portfolio built to showcase my web development projects and technical skills.

<p>Framework: Next.js (Migrated from Vite for better routing and performance)</p>
<p>Styling: Tailwind CSS</p>
<p>Deployment: Vercel</p>
<p>Components: Heroicons / Lucide-React / hand-drawn [icons/images]</p>

Coming from a Vite background, I chose Next.js for this project to learn File-based Routing 
and take advantage of Server-Side Rendering (SSR). This ensures the portfolio is SEO-friendly 
and loads incredibly fast.

//////

<h1>How to run locally</h1>
npm install
<br/>
npm run dev
